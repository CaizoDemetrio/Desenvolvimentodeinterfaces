Visão Geral sobre CSS

Css é um acrônimo para folhas de estilo em cascata. Essa tecnica visa personalisar, esteticar em documento HTML. Para isso, usamos regras CSS, que representam uma estrutura que define como os elementos selecionaveis devem ser estilizados. Essa é a regra é composta pelas seguintes partes:

  • Seletor Indico quais elementos serão afetados personalisarestilo definido.
  • Bloco de Declaração É o bloco composto por propriedades e alores, esse bloco será aplicado ao elemento html.
