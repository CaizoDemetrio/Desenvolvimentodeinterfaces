Exemplo 10 HTML - Tabelas

As tabelas em html são apresentasdas pela tag <table> que são úteis para exibir dados tabulares, tais como informações em linhas colunas.

Descrtição dos principais atributos para criação de tabelas em HTML

  • <>: Esta é a tag principal que envolve toda a tabela html
  • <>: Esta tag representa uma linha da tabela. E nela teremos as tags <th> e <td>
  • <>: Esta tag representa o cabeçalha da tabela, th é o a acrônimo de table header.
  • <>: A tag <td< (table data), representa a celula da tabela.
  • <>: Tag semântica que representa o corpo da tabela, nela geralmente usamos a tag td.
  • <>: Tag semântica que representa o rodapé d tabela.
