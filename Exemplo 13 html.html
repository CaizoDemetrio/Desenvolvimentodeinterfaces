Exemplo 13 HTML - Formularios

Os Formularios HTM sao umas das principais formas de se interagir com aplicações web. Esses formularios ultilizam diversos componentes que melhoraram a esperiencia do usuario

Para trabalhar com formularios devem usar a tag <fom> em conjunto comoutas tags que vão representar os componentes de formularios, tais como: labels(rotulos), inputs(caixa de texto), botões,elementos de checagem, de seleção e entre outros componentes. Neste exemplo, desmonstraremos o uso dos metodos Get e Post

Metodos de Protocolo HTTP

Get

Quando trabalhamos na criação de formularios html, nos deveemos na maior parte das vezes ultilizar o protocolo e redes http, esse protocolo possui varios metodos que são ultilizados para facilitar o processo de comunicação entre cliente e o servidor.

O metodo GET é aquele que anexa os dados do formulario é uma url, esses dados do formulario sõa visiveis na barra de enderço do navegador.

é importante saber que este metodo não recomendado para operações de inclusão, pois os dados vão aparecer para qualquer pessoa, o que pode ser inseguro.

Post

Já o metodo POST é aquele que envia os dados para p servidor no corpo da pagina. Essa informações não são visiveis par o cliente sendo um pouco maus seguras que quando utilizamos o metodo GET.
